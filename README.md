# Retro Scoreboard

A simple scoreboard inspired by retro TV game show "Что? Где? Когда?".

## Demo

A live demo is available here: [https://gdainti.com/chgk-scoreboard/](https://gdainti.com/chgk-scoreboard/)

## How to Use

- Click the top-left corner to increase the left score.
- Click the bottom-left corner to decrease the left score.
- Click the top-right corner to increase the right score.
- Click the bottom-right corner to decrease the right score.
